# LingoLoco

LingoLoco is a Next.js language-learning app built with Supabase for auth, data, and storage.

## Structure

- `app/` for routes, pages, and API handlers
- `components/` for shared UI
- `lib/` for Supabase clients, database helpers, and shared logic
- `public/` for static assets

The codebase already has the right frontend/backend separation for this stack:

- Frontend: `app/`, `components/`
- Backend: `app/api/`, `lib/`

That is the best fit for this project because it keeps deployment simple while still separating UI and server logic.

## Run

```bash
npm run dev
```

## Build

```bash
npm run build
```
